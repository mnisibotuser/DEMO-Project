## 📖 Project Overview

WhatsApp Web Phisher is a demonstration of how QR code phishing (QRLJacking) attacks work. It simulates a fake WhatsApp Web login page that captures phone numbers when users attempt to "verify" their account.

### 🎯 Purpose

- **Educate users** about the dangers of QR code phishing
- **Demonstrate** how social engineering attacks work
- **Raise awareness** about checking linked devices
- **Provide training material** for security professionals

### 🔍 How It Works

1. User visits the fake WhatsApp Web page
2. Page displays a realistic QR code
3. After 30 seconds, asks for phone number "verification"
4. Phone number is captured and stored
5. User is redirected to real WhatsApp Web
